### Nema23 Stepper Motor Controller

ESP32를 통해 Nema23 스테퍼 모터를 제어하는 Qt 기반 GUI 애플리케이션입니다.

### 개발 환경
- **언어**: C++17, Qt 6.9.1
- **라이브러리**: QSerialPort, Qcustomplot
- **아키텍처**: SOLID
